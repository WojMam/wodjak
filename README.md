# WODJAK Gaming — strona linków

Statyczna witryna (GitHub Pages, domena `wodjak.pl`): HTML, CSS, vanilla JavaScript.

- **Motyw:** stały układ ciemny (bez przełącznika jasny/ciemny).
- **Język:** przełącznik PL/EN w pasku nawigacji.
- **Logika:** [`script.js`](script.js) — edytuj ten plik bezpośrednio.

## Zasoby

Katalog `images/` (profile, ikony) musi być obecny lokalnie lub w repozytorium — ścieżki są w HTML.
